attr 이용

  • enabled/disabled 상태 이용가능한 요소

    button, input, seclect, option, textarea

    button {cursor: pointer;}
    button:disabled {opacity:0.8; cursor: initial;}

    input[type="radio"], input[type="checkbox"], option

    input[type="radio"]:checked {outline: 1px dashed red;}

  • 속성

    선택자

    type 선택자 = div,p,h1,span등 태그 선택자
    class 선택자 = .class명으로 지정한 선택자
    id 선택자 = #id명으로 지정한 선택자
    후손 선택자 div p = div 의 자식 (모든 p)
    직계 선택자 div > p = div 의 자식 (1대 p)
    인접 선택자 div + p = div 의 형제 p (div 바로 뒤에 오는 형제 p)
    형제 선택자 div ~ = div 다음에 오는 모든 형제 요소
    수열 선택자 div:first-child/nth-child(3)/last-child/last-of-type
    속성 선택자 img[alt] alt속성을 가진 img 요소
    반응 선택자 :hover/checked/focus

    display

    inline | block | flex | inline-block | inline- flex | inline-table | list-item | run- in | tabl e |table-caption | table-column- group | table-header-group | table-footer- group | table-row-grou p |table-cell | table- column | table-row | none

    animation

    animation-name: @keyframes 이름 (예시에서는 tutsFade를 사용함)
    animation-duration: 타임프레임 길이. 애니메이션 시작부터 마지막까지 총 지속시간
    animation-timing-function: 애니메이션 속도 조절 ( linear | ease | ease-in | ease-out | ease-in-out | cubic-bezier )
    animation-delay: 애니메이션이 시작하기 전 지연시간
    animation-iteration-count: 반복 횟수
    animation-direction: 루프 (loop) 방향. 정방향으로 반복, 역방향으로 반복, 번갈아가며 반복 등을 설정
    animation-fill-mode: 애니메이션 시작/끝 상태 제어 ( none | forwards | backwards | both )

    table에 scroll을 적용해야한다면
    table(thead~tfoot) 전체 scroll은 css로 적용이 가능

    만약, tbody만 scroll을 적용해야한다면

    1. css로 구현해야 할 경우 (접근성 불충족)

    시멘틱 마크업

    의미에 맞는 tag를 사용함으로써 tag만 보아도 제목, 내용의 구분이 확실하게 느껴지게 하는 것
